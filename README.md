# 🍴 The Golden Fork - Restaurant Menu

A modern, responsive restaurant menu and reservation page built with HTML5 and CSS3. 

## 🚀 Live Demo
(https://my-restaurant-menu-bice.vercel.app/)


## ✨ Features
- **Responsive Design:** Works on mobile, tablet, and desktop.
- **Interactive Menu:** Grid layout with hover effects.
- **Reservation Form:** Clean, user-friendly booking interface.
- **Fast Loading:** Optimized for performance using CSS Grid and Flexbox.

## 🛠️ Tech Stack
- HTML5
- CSS3 (Custom Properties & Grid)
- Unsplash API (for high-quality imagery)
